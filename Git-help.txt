$ git init - Эта команда создаёт в текущем каталоге 
			новый подкаталог с именем .git содержащий все необходимые 
			файлы репозитория — основу Git-репозитория.
			
$ git clone https://github.com/FujiF/Test_repo.git - Клонирование репозитория
из сети

$ git add - индексирует файлы в рабочего каталога

$ git commit -a -m(message 'hello')- заставляет Git автоматически индексировать
каждый уже отслеживаемый на момент коммита файл, позволяя вам обойтись без git add.

$ git rm - удаляет файл из вашего рабочего каталога

$ git rm --cached readme.txt -  вы можете захотеть оставить файл на винчестере,
 и убрать его из-под бдительного ока Git'а
 
$ git mv file_from file_to - переименовать файл

$ git log (--stat - более подробно)- выводит список коммитов созданных в данном 
									 репозитории
									 
$ git log --pretty=format:"%h - %an, %ar : %s"
ca82a6d - Scott Chacon, 11 months ago : changed the version number
Параметр    Описание выводимых данных
%H  Хеш коммита %h  Сокращённый хеш коммита
%T  Хеш дерева %t  Сокращённый хеш дерева
%P  Хеши родительских коммитов %p  Сокращённые хеши родительских коммитов
%an Имя автора %ae Электронная почта автора
%ad Дата автора (формат соответствует параметру --date= )
%ar Дата автора, относительная (пр. "2 мес. назад")
%cn Имя коммитера %ce Электронная почта коммитера
%cd Дата коммитера %cr Дата коммитера, относительная
%s  Комментарий

$ gitk - Визуализация истории (forgot)
